## [0x00] OVERVIEW

Desarrollador de software y analista de sistemas con un enfoque riguroso en la lógica del lado del servidor, diseño de bases de datos y despliegue de arquitecturas escalables. Mi metodología de trabajo se basa en la comprensión profunda de cómo operan los sistemas a bajo y alto nivel, priorizando el rendimiento, la seguridad y las buenas prácticas en el control de versiones (flujos avanzados en Git/GitHub).

Actualmente cursando el Tecnólogo ADSO, liderando el desarrollo técnico de proyectos de alta complejidad que involucran múltiples interfaces, roles y topologías de red.
